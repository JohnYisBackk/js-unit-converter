# 🔄 Unit Converter PRO

Modern and clean unit converter built with **HTML, CSS and JavaScript**.  
Supports multiple categories with a smooth and user-friendly interface.

---

## 🚀 Features

- Convert between **Length, Weight and Temperature**
- Clean and modern UI design
- Swap units instantly
- Copy result to clipboard
- Input validation
- Responsive design (works on mobile)
- Dynamic unit selection based on category

---

## 🧠 Technologies Used

- HTML5
- CSS3 (Custom properties / variables)
- Vanilla JavaScript (no frameworks)

---

## 📐 Supported Conversions

### Length
- Meter
- Kilometer
- Centimeter

### Weight
- Gram
- Kilogram
- Pound

### Temperature
- Celsius
- Fahrenheit
- Kelvin

---

## ⚙️ How It Works

1. Select conversion category
2. Choose units (from → to)
3. Enter value
4. Click **Convert** or press **Enter**
5. Get instant result
